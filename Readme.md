# DevConf 2026 — Landing Page

A single-page developer conference landing site built with pure HTML5 and CSS3.

## Tech Stack
- HTML5
- CSS3 (no frameworks, no libraries)
- No JavaScript

## Sections
1. **Nav** — logo, links, register button
2. **Hero/Banner** — dark image background with headline and CTA
3. **Meet the Speakers** — 2x2 card grid
4. **Secure Your Spot** — 3-column pricing cards with one featured dark card
5. **Highlights From 2025** — stat band (attendees, speakers, countries, talks) + quote band
6. **Footer** — two-row layout with social links and site links

## AI Usage
The **Highlights From 2025** section was ideated, designed, and coded with AI assistance (Claude), as permitted by the assignment. This included brainstorming section ideas, choosing a layout direction, generating the base HTML structure, and writing the CSS to match the existing design system (colors, spacing, typography). All prompts used are included in `PROMPTS.md`.

All other sections were built independently.
